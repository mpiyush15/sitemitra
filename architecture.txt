Site Mitra — System Architecture & Technical Planning
1. Project Overview
Site Mitra is a construction and property business networking platform designed to connect professionals, vendors, suppliers, builders, and customers into one digital ecosystem.
The platform functions as:
Business Listing Platform
Vendor Showcase Platform
Lead Generation Platform
Networking Platform
Membership-Based Visibility Platform

4. User Roles & RBAC Planning
RBAC (Role-Based Access Control)
1. Super Admin
Permissions:
Full system control
Manage users
Manage plans
Manage payments
Manage listings
Manage categories
Moderate reviews
Control homepage sections
Access analytics

2. Business User (Free)
Permissions:
Create profile
Edit profile
Basic listing visibility
Add contact information
WhatsApp integration
View dashboard
Restrictions:
No premium visibility
No reviews
No project gallery
No catalogue upload

3. Business User (Standard/Premium)
Permissions:
Featured listing visibility
Upload project gallery
Upload catalogues
Display reviews/ratings
Verification badge
Better search visibility
Lead access
Vendor showcase

4. Customer/Visitor
Permissions:
Search listings
View profiles
Send inquiries
Contact businesses
Download catalogues
